Fork of DownItAll, which is a fork of version 3.0.8 of DownThemAll!, the download manager for Firefox. Retains all the features and browser integration that the current WebExtension version for Firefox doesn't. 

How to install: Code button, Download ZIP, drag downloaded zip into browser.
If it refuses to install, extract zip, and recompress to zip from the directory where all the files are in.
